# Patriot Store

Преміальний інтернет-магазин пневматичної зброї, статуеток, Zippo та засобів самооборони.

## Локальний запуск

```bash
python -m http.server 8080
```

Відкрийте [http://localhost:8080](http://localhost:8080)

## Стек

- HTML + Tailwind CSS
- Vanilla JavaScript
- Liquid Glass design system
